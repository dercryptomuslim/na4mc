# NA4MC - Nomad Agency for Marketing & Consulting LLC
## Webseite-Konzept & Brandbook

Dieses Dokument enthält die komplette Struktur, alle Texte, das Design-System und die Implementierungsanleitung für die neue Webseite von NA4MC.

---

### 1. Brand Identity & Design System

**Brand Vibe:** "Modern – Nomadic – Digital – Muslim – Premium – Minimalistisch"

#### Farbpalette
Das Farbschema kombiniert warme Wüstentöne mit professionellem Dark Mode und Premium-Akzenten.

| Farbe | Hex Code | Verwendung |
| :--- | :--- | :--- |
| **Sand Beige** | `#E6DACE` | Hintergrund Sekundär, Karten-Highlights, Weiche Texte |
| **Desert Orange** | `#C25E00` | CTA Buttons, Akzente, Icons (Warm, Energetisch) |
| **Deep Night Blue** | `#0B1120` | Haupt-Hintergrund, Footer (Professionell, Seriös) |
| **Charcoal Black** | `#1A1A1A` | Karten-Hintergrund, Texte auf hellen Flächen |
| **Gold Premium** | `#D4AF37` | Feinste Linien, Borders, Premium-Badges |
| **Soft White** | `#F5F5F5` | Haupttext auf dunklem Hintergrund |

#### Typografie
*   **Headlines:** *Inter* oder *Plus Jakarta Sans* (Bold, Tight Tracking) - Modern, Digital.
*   **Body:** *Inter* oder *DM Sans* (Regular) - Sehr gut lesbar.

#### UI Komponenten (shadcn/ui & Tailwind Stil)
*   **Buttons:** Abgerundet (Rounded-full oder md), Hover-Effekte mit Glow (Gold/Orange).
*   **Cards:** Dunkles Glas-Design (Glassmorphism) mit feinem Gold/Beige Border.
*   **Icons:** Thin Line Icons (Lucide React), Gold oder Orange eingefärbt.

### 4. Technische Implementierung (Next.js + Tailwind)

Die folgende Ordnerstruktur und Code-Basis wird für das Projekt erstellt:

*   **Framework:** Next.js 14 (App Router)
*   **Styling:** Tailwind CSS
*   **Components:** shadcn/ui (Radix UI Primitive)
*   **Icons:** Lucide React
*   **Fonts:** Inter & Playfair Display (für Akzente)

Dieses Konzept ist bereit zur direkten Umsetzung im Code.
